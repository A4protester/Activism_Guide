# 示威者参与指南汇总
##### Q: 我为什么要花时间看这些文章？

##### A:

##### 1. 为自己也是为同伴。花几小时阅读，换来的可能是自己或同伴几年的自由， 甚至是生命。 很值。
##### 2. 为了抗争，这会更大程度瘫痪维稳系统，抗争成功机率会更大。
##### 3. Something is better than nothing. 有点准备永远胜于毫无准备。


##### Q: 这些文章里的内容现在似乎还用不上，只有在相当激烈的时候才有用。我为什么要看？

##### A:为的是以后可能更大的抗争。
